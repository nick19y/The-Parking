As regras a seguir dizem um pouco a respeito da codificação no projeto com o objetivo de padronizar o código

Variáveis:
devem ser escritas descrevendo um pouco do que cada uma faz
devem ser escritas com letra minúscula inicialmente e se necessária a utilização de outra palavra, escrita com letra maiúscula como no exemplo a seguir:


btnCriarConta

A nomeação de pastas deve obedecer as regras acima 

CSS:
as configurações padrão do projeto encontram-se no arquivo styles.css
o arquivo reset css foi usado para padronizar o projeto na tela de cada navegador
para a estilização de cada página especifica deve ser criada uma pasta dentro da pasta css que indique o arquivo contido nela

IMG:
quando for necessário usar imagens para o projeto, deve-se verificar os direitos de uso de imagem e creditá-las no arquivo de créditos na raíz do projeto
